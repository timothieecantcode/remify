# REMify

A simple and intuitive sleep cycle calculator that helps users determine the best times to sleep or wake up based on complete sleep cycles.

## Features

- Calculate optimal wake-up times based on bedtime
- Calculate ideal bedtime based on wake-up time
- “Sleep Now” mode using current time
- Supports multiple cycle lengths (75, 90, 105 minutes)
- Highlights recommended sleep durations (7–9 hours)

## Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS

As a software engineering student, I wanted to build a practical tool to improve sleep quality after long study days. This project focuses on clean UI, user-friendly interaction, and real-world usability.
